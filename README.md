# [Installation using script]
* Masakari Installation usiing script 1st need to clone the repository
```bash
$ sudo -s
# git clone https://github.com/vichuvishnu/mdcMasakari
# cd mdcMasakari
```
* Edit the local.conf file
```bash
# vi local.conf
```
* Now the script is ready to run 
```bash
# ./masakari.sh
```
